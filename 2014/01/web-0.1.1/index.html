Released Earth Reader for Web 0.1.1

We released Earth Reader for Web 0.1.1: This version fixes several crawler bugs; added few commandline options.

GitHub
https://github.com/earthreader/web/releases/tag/0.1.1
PyPi
https://pypi.python.org/pypi/EarthReader-Web/0.1.1

Use pip to install or upgrade:

$ pip install EarthReader-Web==0.1.1

The Earth Reader team and other contributors put many efforts into this. Thanks for Choi Bochul, Hong Minhee, and Yong Choi. This is Kang Jinwon.

Thanks,
Jinwon and Earth Reader team
